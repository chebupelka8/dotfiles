```python
array = [2, 1, 3, 2, 5]
array[3]  # Получение элемента по индексу: O(1)
```

```python
array = [2, 1, 3, 2, 5]
array.append(6)  # Добавление в динамический массив: в лучшем случае O(1), в худшем O(n)
```

```python
array = [4, 4, 1, 1, 1, 5, 3, 3]
set(array)  # Преобразование списка во множество: O(n)
```
